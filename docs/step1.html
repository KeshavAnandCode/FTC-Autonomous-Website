Step 1: Name and Package

Enter Autonomous Program Name

Enter any name. This will be both the name of the class and the name of the autonomous program. You must follow java naming rules (no spaces, etc.)

Enter Package (Folder) Name

This is the path to the folder where you will store your autonomous program. In Android Studio, you can easily find the package by right clicking the desired folder and clicking 'Copy Path/Reference'.

Then, click 'Copy Reference' to get your package.
